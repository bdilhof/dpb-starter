# DPB Starter Project

## System requirements

-   WSL2
-   Docker
-   Git
-   Composer

## Commands

```
composer create-project --repository '{"type": "vcs", "url": "https://github.com/bdilhof/dpb-starter"}' --stability=dev --ignore-platform-reqs bdilhof/dpb-starter testovacka
```

Open http://localhost
